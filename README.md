# Aamir AWS Portfolio

A modern cloud portfolio website built with React, Vite, and AWS Amplify.

## Live Demo

https://main.d1fj9kxl1oyawh.amplifyapp.com/

## Features

- Responsive portfolio website
- Project showcase section
- Clean modern UI
- Fast Vite build
- AWS Amplify deployment
- Contact section

## Tech Stack

- React
- Vite
- JavaScript
- CSS
- AWS Amplify
- GitHub

## Run Locally

npm install
npm run dev

## Build

npm run build
